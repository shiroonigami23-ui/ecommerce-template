# E-Commerce Template

Full-stack e-commerce template.

## Quick Start
git clone https://github.com/shiroonigami23-ui/ecommerce-template.git
cd ecommerce-template
npm run dev

Frontend: http://localhost:3000
Backend: http://localhost:5000/api

## Add GitHub Secrets:
https://github.com/shiroonigami23-ui/ecommerce-template/settings/secrets/actions

Add: DOCKERHUB_USERNAME, DOCKERHUB_TOKEN, VERCEL_TOKEN
